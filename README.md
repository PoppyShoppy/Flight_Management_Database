# Flight_Management_Database

A simple CLI app for managing flights, destinations, pilots, and crew assignments using SQLite.

## Features

- Create tables and seed sample data
- Insert, search, update, and delete records
- View summaries and statistics

## Requirements

- Python 3.10+ (tested with system Python)

## Run

```bash
python3 main.py
```

## First run

Choose menu option `1` to create the tables and seed sample data. After that, you can use options 2-7 for CRUD and reports.

## Notes

- The database file is `DATABASE.db` in the project root.
- If you delete the database file, run option `1` again to recreate it.
